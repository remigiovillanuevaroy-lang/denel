# Denel Perú - Tienda de Cacao

Página web de productos de cacao para Denel Perú.

## Estructura del Proyecto

```
chocolate/
├── index.html          # Página principal
├── images/             # Carpeta para imágenes de productos
└── README.md           # Este archivo
```

## Imágenes Requeridas

Para que la página se vea completa, necesitas agregar las siguientes imágenes a la carpeta `images/`:

1. **cacao-polvo.jpg** - Imagen del producto "Cacao en Polvo"
2. **cacao-grano.jpg** - Imagen del producto "Cacao en Grano"
3. **cacao-barra.jpg** - Imagen del producto "Cacao en Barra 100% Puro"
4. **cacao-pelado-tostado.jpg** - Imagen del producto "Cacao Pelado y Tostado"
5. **pasta-cacao.jpg** - Imagen del producto "Pasta de Cacao Natural"
6. **nibs-cacao.jpg** - Imagen del producto "Nibs de Cacao"

**Nota:** Las imágenes pueden estar en formato JPG, PNG, o cualquier otro formato de imagen compatible. Solo asegúrate de que los nombres de archivo coincidan exactamente con los listados arriba.

## Cómo Usar

1. Coloca tus imágenes de productos en la carpeta `images/`
2. Asegúrate de que los nombres de archivo coincidan con los especificados arriba
3. Abre `index.html` en tu navegador para ver la página
